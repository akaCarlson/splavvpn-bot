iPhone / iOS — настройка клиента (AmneziaWG / AmneziaVPN) Важно про РФ В российском App Store приложение AmneziaVPN может быть недоступно. Официальные варианты: сменить регион/создать Apple ID другого региона или использовать альтернативное приложение. Что понадобится 1) Клиент на iOS: • AmneziaWG (если AmneziaVPN недоступен) • или AmneziaVPN (если установлен/доступен) 2) Ключ от бота: • Текстовый ключ: /request_text • Файл: /request_config • QR: /request_qr Вариант 1 (рекомендуется): подключение по текстовому ключу 1) В боте нажми: /request_text 2) Скопируй ключ в буфер обмена (ключ — это всё, что внутри кодового блока) ‼️Для настройки первого сервера в клиенте: 3) Открой AmneziaWG/AmneziaVPN → нажми «Приступим» → выбери «Текстовый ключ» 4) Вставь ключ → нажми «Подключиться» → дождись настройки → затем нажми «Подключиться» на главном экране ‼️Если в клиенте уже есть сервера и ты хочешь добавить новый: 3) На главном экране нажми «+» → выбери «Текстовый ключ» 4) Вставь ключ → «Подключиться» → дождись настройки → «Подключиться» ──────────────────────────── Вариант 2: подключение по QR (удобно для настройки на другом устройстве) 1) В боте нажми: /request_qr ‼️Для первого сервера: 2) Открой клиент → «Приступим» → «QR-код» :contentReference[oaicite:4]{index=4} 3) Сканируй QR (если несколько — по очереди) 4) На предпросмотре нажми «Подключиться» → дождись настройки → «Подключиться» на главном экране ‼️Если сервера уже есть: 2) На главном экране «+» → «QR-код» 3) Сканируй → «Подключиться» ──────────────────────────── Вариант 3: подключение по файлу 1) В боте нажми: /request_config (получишь файл .conf, сохрани его в «Файлы») ‼️Для первого сервера: 2) Открой клиент → «Файл настроек подключения» → выбери файл 3) Посмотри содержимое → «Подключиться» → дождись настройки → «Подключиться» ‼️Если сервера уже есть: 2) На главном экране «+» → «Файл настроек подключения» → выбери файл 3) «Подключиться» → дождись настройки → «Подключиться» ──────────────────────────── Если не подключается • Открой /status и проверь, что профиль в панели есть и не отозван • Попробуй заново получить файл: /request_config
