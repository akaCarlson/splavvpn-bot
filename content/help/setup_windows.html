Windows — настройка клиента (AmneziaVPN / AmneziaWG) Что понадобится 1) Клиент на Windows: • AmneziaVPN (основной) • или AmneziaWG (если удобнее/нужна совместимость) 2) Ключ от бота: • Текстовый ключ: /request_text • Файл: /request_config • QR: /request_qr Вариант 1 (рекомендуется): подключение по файлу 1) В боте нажми: /request_config (получишь файл .conf, сохрани его на ПК) ‼️Для настройки первого сервера в клиенте: 2) Открой AmneziaVPN/AmneziaWG → нажми «Приступим» 3) Выбери «Файл настроек подключения» → укажи скачанный .conf 4) Посмотри содержимое → «Подключиться» → дождись настройки → «Подключиться» ‼️Если сервера уже есть и нужно добавить новый: 2) На главном экране нажми «+» → «Файл настроек подключения» → выбери файл 3) «Подключиться» → дождись настройки → «Подключиться» ──────────────────────────── Вариант 2: подключение по текстовому ключу 1) В боте нажми: /request_text 2) Скопируй ключ (всё из кодового блока) 3) В клиенте выбери «Текстовый ключ» 4) Вставь → «Подключиться» → дождись настройки → «Подключиться» ──────────────────────────── Вариант 3: QR (если есть камера/удобно сканировать) 1) В боте: /request_qr 2) В клиенте: «QR-код» → сканируй → «Подключиться» ──────────────────────────── Если не подключается • Открой /status и проверь, что профиль в панели есть и не отозван • Пересоздай/переполучи файл: /request_config • Если есть ошибка в клиенте — сделай скрин и отправь админу
