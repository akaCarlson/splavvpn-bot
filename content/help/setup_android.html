Android — настройка AmneziaVPN Что понадобится 1) Приложение AmneziaVPN из магазина приложений 2) Ключ от бота: • Текстовый ключ: /request_text • Файл: /request_config • QR: /request_qr Вариант 1 (рекомендуется): подключение по текстовуму ключу 1) В боте нажми: /request_text 2) Скопируй ключ в буфер обмена (ключ - это все, что входит в кодовый блок) ‼️Для настройки первого сервера в клиенте: 3) Открой AmneziaVPN → на первом экране нажми «Приступим» → выбери «Текстовый ключ» 4) Вставь ключ в поле → нажми «Подключиться», дождись настройки, затем нажми «Подключиться» на главном экране ‼️Если в клиенте уже есть сервера и ты хочешь добавить новый, то: 3) На главном экране AmneziaVPN нажми «+» → выбери «Текстовый ключ» 4) Вставь ключ в поле → «Подключиться» → дождись настройки → «Подключиться» ──────────────────────────── Вариант 2: подключение по QR (удобно для настройки VPN на другом устройстве) 1) В боте нажми: /request_qr ‼️Для настройки первого сервера в клиенте: 2) Открой AmneziaVPN → на первом экране нажми «Приступим» → выбери «QR-код» 3) Наведи камеру на QR-код (если QR-кодов несколько — сканируй по очереди) 4) На экране предпросмотра нажми «Подключиться», дождись настройки, затем нажми «Подключиться» на главном экране ‼️Если в клиенте уже есть сервера и ты хочешь добавить новый, то: 2) На главном экране AmneziaVPN нажми «+» → выбери «QR-код» 3) Наведи камеру на QR-код (если QR-кодов несколько — сканируй по очереди) 4) На экране предпросмотра нажми «Подключиться», дождись настройки, затем нажми «Подключиться» на главном экране ──────────────────────────── Вариант 3: подключение по файлу 1) В боте нажми: /request_config (получишь файл .conf, сохрани его на устройстве) ‼️Для настройки первого сервера в клиенте: 2) Открой AmneziaVPN → «Файл настроек подключения» → выбери скачанный файл 3) Посмотри содержимое → нажми «Подключиться» → дождись настройки → «Подключиться» ‼️Если в клиенте уже есть сервера и ты хочешь добавить новый, то: 2) На главном экране AmneziaVPN нажми «+» → выбери «Файл настроек подключения» → выбери скачанный файл 3) Посмотри содержимое → нажми «Подключиться» → дождись настройки → «Подключиться» ──────────────────────────── Если не подключается • Открой /status и проверь, что профиль в панели есть и не отозван • Попробуй заново получить файл: /request_config
